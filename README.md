# 📊 Análise Comercial com SQL

## 🎯 Objetivo

Realizar análise de dados de vendas utilizando SQL, com foco em identificar faturamento, desempenho de clientes e comportamento de produtos.

## 💼 Contexto

Simulação de um cenário empresarial onde uma empresa precisa analisar seus dados comerciais para apoiar decisões estratégicas.

## 🧱 Estrutura do Banco

O projeto foi estruturado com três tabelas principais:

* clientes → dados dos clientes
* produtos → informações dos produtos
* vendas → registros de transações

## 🛠️ Técnicas Utilizadas

* JOIN → relacionamento entre tabelas
* GROUP BY → agregação de dados
* SUM → cálculo de faturamento
* CASE WHEN → classificação de dados
* LEFT JOIN → identificação de ausência de dados

## 📈 Análises Realizadas

* Faturamento total
* Faturamento por cliente
* Produto mais vendido
* Identificação de clientes sem compra
* Classificação de clientes por desempenho

## 📊 Insights

* Clientes Carlos e Ana apresentaram maior faturamento no período
* João apresentou menor desempenho
* Possibilidade de campanhas para clientes com baixo faturamento
